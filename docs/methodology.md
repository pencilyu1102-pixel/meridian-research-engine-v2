# Methodology

Meridian Research Engine 2.0 follows a fixed order in the `v2.0.0-beta` initial release:

```text
Macro cycle
-> Asset rotation
-> Industry position
-> Company fundamentals
-> Data reliability audit
-> Data meaning explanation
-> Valuation and margin of safety
-> Price Level Engine
-> Real portfolio execution
-> Bear Case
-> Team Lead synthesis
-> Report gatekeeping
```

## Four-Layer Judgment

Every final report must separate:

1. company judgment;
2. valuation judgment;
3. account judgment;
4. trade judgment.

This avoids turning "good company" into "good buy" without valuation and account checks.

## Decision Weight

Decision weight should reflect real influence on the conclusion. A variable that is uncertain but decisive should be ranked higher than a precise but low-impact metric.

## Bear Case

The Bear Case Agent must attack the main conclusion before the report is finalized. If the strongest opposing evidence is material and unresolved, the report should be downgraded or blocked.

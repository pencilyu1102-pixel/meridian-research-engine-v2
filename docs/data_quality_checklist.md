# Data Quality Checklist / 数据质量检查清单

## Purpose

标准化数据质量检查清单。每份报告在准出前必须完成本清单。本清单防止最常见的研究错误：股价过期、币种混乱、股本未更新、会计口径混用。

## 一、股价与市值数据

| 检查项 | 通过/失败 | 备注 |
|---|---|---|
| 股价日期与报告日期一致 | | |
| 市值使用最新股本计算 | | |
| 是否发生拆股、送转、增发、ADS比例变化、配股 | | |

## 二、估值口径

| 检查项 | 通过/失败 | 备注 |
|---|---|---|
| PE/PS/EV/EBITDA — 使用的是 TTM、Forward 还是历史值？ | | |
| Forward PE 用的是 FY1 还是 FY2 的共识？ | | |
| EV 计算是否使用最新净债务？ | | |

## 三、币种与单位

| 检查项 | 通过/失败 | 备注 |
|---|---|---|
| 所有数据是否使用统一币种？ | | |
| 美股是否区分 GAAP 和 Non-GAAP？ | | |
| 港股是否区分港币、人民币和美元收入？ | | |
| 全球股票是否存在汇率换算影响？ | | |
| Non-GAAP 调整项是否已披露原因？ | | |

## 四、利润表调整项

| 检查项 | 通过/失败 | 备注 |
|---|---|---|
| 一次性收益或费用是否已识别？ | | |
| 资产减值和存货跌价是否已标注？ | | |
| 政府补助是否已识别？ | | |
| 股权激励（SBC）是否已调整？ | | |
| 税率异常是否已标注？ | | |

## 五、同口径可比性

| 检查项 | 通过/失败 | 备注 |
|---|---|---|
| 同业对比是否使用相同指标口径？ | | |
| 历史数据是否使用相同会计准则？ | | |
| 并购或剥离导致的并表范围变化是否已标注？ | | |

## 六、后果规则

如果任何关键数据（股价、EPS、市值）过期或错误，报告准出状态必须降级为：
- **条件准出**（不影响核心判断的数据过期）
- **不可准出**（关键数据错误或估值口径混乱）

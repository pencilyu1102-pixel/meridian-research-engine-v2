# Implied Expectation Model / 市场隐含预期模型

## Principle
Before setting a target price, reverse-engineer what the current price already discounts. A stock that looks 'cheap' may already be pricing in deteriorating fundamentals.

## 1. What the Current Price Implies

| Assumption | Market Implied | Our View | Gap | Evidence |
|---|---|---|---|---|
| Future Revenue CAGR | | | | |
| Long-term Gross Margin | | | | |
| Long-term Net Margin / FCF Margin | | | | |
| Market Share | | | | |
| Valuation Multiple (PE/EV/Sales) | | | | |
| ROIC / ROE | | | | |

**Key questions:**
- What future CAGR does price discount?
- Consistent with TAM and competition?
- What margin / share does price imply?
- Do we believe it?
  - If yes, is upside sufficient?
  - If no, how large is downside?

## 2. Valuation Method Selection Table

| Company Type | Primary Method |

| Cyclicals | EV/EBITDA mid-cycle, P/B, Cost curve |
| Banks | P/B, ROE, Dividend yield |
| Insurance | P/EV, P/B, New business value |

| Biotech | rNPV, Pipeline probability-weighted |
| Internet platform | EV/EBITDA, FCF, User value |
| Consumer brands | PE, EV/EBITDA, ROIC, Brand premium |

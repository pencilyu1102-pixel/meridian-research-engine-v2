# Archived Non-Public Case Studies / 非公开历史案例

This directory stores historical company-specific research outputs used only for internal process review and framework validation.

本目录仅用于保存历史性的具体公司研究样例，用于内部流程复盘、结构校验、数据审计和 Gatekeeper 测试参考。

These files are not public sample reports and should not be used as investment advice, trading recommendations, price forecasts, target-price guidance, or return expectations.

这些文件**不是公开展示样例**，不构成投资建议、买卖推荐、股价预测、目标价指引或收益承诺。

They may contain company-specific public information, consensus data, valuation assumptions, analyst target-price references, or scenario analysis. Such content is retained only to demonstrate how the research workflow handles real-world complexity.

其中可能包含具体公司的公开资料、市场共识、估值假设、分析师目标价引用或情景分析。这些内容仅用于说明研究流程如何处理真实世界的复杂信息。

For public launch and external communication, use only generic examples such as SAMPLE, TICKER, or ABC.

首发传播和外部展示时，仅使用 `SAMPLE`、`TICKER` 或 `ABC` 等虚拟示例。

Do not publish, quote, or promote files in this directory as stock recommendations or model output examples.

不得将本目录文件作为荐股内容、交易信号、投资结论或模型输出样例进行公开传播。

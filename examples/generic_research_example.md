# Generic Research Example

This example uses fictional tickers and fictional assumptions. It is educational only and is not financial advice.

## One-Sentence Framing

ABC requires separate judgments for company quality, valuation range, account suitability, and execution conditions.

## Key Workflow

1. Macro & Liquidity Agent reviews whether the macro environment supports the relevant valuation framework.
2. Asset Rotation Agent checks whether the target sector has broad or narrow market support.
3. Industry Position Agent separates industry beta from company alpha.
4. Company Fundamental Agent reviews revenue quality, profit quality, cash-flow quality, capital allocation, and management.
5. Data Auditor Agent checks source quality, timestamp, unit, currency, and accounting basis.
6. Valuation & Margin Agent builds generic scenario assumptions.
7. Price Level Engine maps valuation anchors.
8. Portfolio Execution Agent checks cost basis, cash, concentration, and drawdown capacity.
9. Bear Case Agent attacks the main conclusion.
10. Team Lead Agent preserves conflicts and produces conditional research conclusions.

## Boundary

No example in this repository should be treated as a recommendation to buy, sell, hold, subscribe, redeem, hedge, or trade any real security.

# Request Examples

## Stock Research

```text
Use Meridian Research Engine 2.0 to review a fictional company named ABC.
Follow SKILL.md, include Data Meaning Blocks, Bear Case, Price Level Engine, and Report Gatekeeper.
Do not provide financial advice or automatic buy/sell recommendations.
Do not include real holdings, real cost basis, or real transaction actions in public examples.
```

## Earnings Review

```text
Use workflows/earnings_review.md to review the latest earnings release.
Separate data facts, data meaning, valuation impact, and account constraints.
```

## Portfolio Review

```text
Use examples/transactions_example.csv and examples/holdings_example.csv.
Calculate management cost, review position weight, and output conditional review triggers.
Treat all data as fictional sample data.
```

## Data Audit

```text
Audit each key data point with source tier, timestamp, unit, currency, and accounting basis.
Downgrade the conclusion if material data cannot enter the report.
```

# Macro & Sector Rotation Workflow

入口：`/macro-sector-rotation`

用途：将宏观共识、市场价格验证和板块轮动判断整理成可审计的研究背景。该 workflow 不输出公开买卖建议，不展示真实个人持仓或真实个股操作。

## 1. 适用场景

- 做单公司研究前，先判断宏观与板块背景。
- 做组合复盘时，判断仓位暴露是否集中在顺风或逆风板块。
- 做行业漏斗筛选时，识别哪些板块值得进入下一层研究。
- 做月度或季度复盘时，检查上一期宏观共识是否已经被市场 price in。

## 2. 输入要求

- 研究市场：A 股 / 港股 / 美股 / 全球市场。
- 研究周期：weekly / monthly / quarterly。
- 官方或准官方数据与政策更新。
- 机构宏观、资产配置或行业配置观点。
- 市场价格和高频指标。
- 待观察板块或行业列表。

## 3. 执行顺序

```text
宏观共识收集
→ 六因子宏观评分
→ 市场价格验证
→ 板块轮动判断
→ 三道门检查
→ 风险与失效条件
→ 交给行业 / 公司 / 估值 / 持仓执行模块
```

## 4. 必须收集的三类信息源

至少覆盖三类来源：

| 来源层 | 最低要求 | 目的 |
|---|---:|---|
| 官方或准官方 | 1 个 | 确认政策、增长、通胀、信用和流动性大方向 |
| 主流机构 | 2 个 | 识别机构共识、分歧和边际变化 |
| 市场价格 | 3 个 | 验证共识是否已被资金行为反映 |

每条来源整理为 `templates/macro_view_card_template.md`。

## 5. 六因子评分步骤

使用 `templates/macro_six_factor_score_template.md` 或 `tools/macro_score.py`。

六因子：

1. 增长；
2. 通胀；
3. 流动性；
4. 信用；
5. 盈利；
6. 风险偏好。

每个因子从 -2 到 +2 评分，并写明证据、来源等级和失效条件。

## 6. 机构共识与市场定价对照

使用 `templates/market_pricing_assessment_template.md` 判断：

- 已充分反映；
- 部分反映；
- 尚未充分反映；
- 可能过度反映。

必须说明依据：价格表现、估值分位、成交量、资金流、市场宽度、拥挤度或相关高频指标。

## 7. 板块轮动判断

使用 `templates/sector_rotation_template.md` 评估：

- 哪些板块获得宏观顺风；
- 哪些板块被宏观压制；
- 行业景气是否确认宏观逻辑；
- 机构共识是否已被市场 price in；
- 交易是否拥挤。

## 8. 三道门检查

| 门槛 | 必答问题 |
|---|---|
| 第一门：宏观风向 | 宏观是否支持该板块？ |
| 第二门：行业景气 | 订单、价格、库存、利润率是否改善？ |
| 第三门：估值与交易拥挤度 | 估值是否合理？交易是否拥挤？ |

三道门结果只能决定研究优先级，不能直接生成公开买卖指令。

## 9. 输出模板

```markdown
# 宏观与板块轮动报告

## 1. 一句话结论

## 2. 宏观共识与分歧

## 3. 六因子宏观评分

## 4. 市场价格验证

## 5. 板块轮动地图

## 6. 三道门检查

## 7. 顺风板块与逆风板块

## 8. 风险清单

## 9. 后续跟踪指标

## 10. 失效条件
```

## 10. 禁止事项

- 不得只引用单一机构就形成结论。
- 不得把机构观点当成确定事实。
- 不得跳过市场价格验证。
- 不得忽略估值和交易拥挤度。
- 不得输出真实个人持仓、真实成本、真实加减仓动作或具体公开买卖点。

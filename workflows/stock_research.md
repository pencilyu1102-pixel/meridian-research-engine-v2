# Stock Research Workflow

## Purpose

Build a full investment research report without turning it into financial advice or an automatic trade recommendation.

## Required Order

1. Macro & Liquidity Agent
2. Asset Rotation Agent
3. Industry Position Agent
4. Company Fundamental Agent
5. Data Auditor Agent
6. Valuation & Margin Agent
7. Price Level Engine
8. Portfolio Execution Agent
9. Bear Case Agent
10. Team Lead Agent
11. Report Gatekeeper

## Required Artifacts

- Data Meaning Blocks.
- Decision Weight Table.
- Core Variable Ranking.
- Four-Layer Judgment.
- Price Level Engine map.
- Bear Case.
- Report Gatekeeper result.

## Output Rule

The final report must distinguish company judgment, valuation judgment, account judgment, and trade judgment.

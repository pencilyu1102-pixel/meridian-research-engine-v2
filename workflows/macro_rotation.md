# Macro Rotation Workflow

## Purpose

Connect macro conditions with asset rotation before studying individual companies.

## Steps

1. Macro & Liquidity Agent classifies macro as tailwind, neutral, or headwind.
2. Asset Rotation Agent identifies where capital is flowing.
3. Industry Position Agent checks which industries benefit or suffer.
4. Valuation & Margin Agent judges whether multiple expansion is supported.
5. Team Lead Agent updates research stance and risk budget.

## Output

- Macro state.
- Rotation map.
- Supported and unsupported sectors.
- Implications for valuation tolerance.

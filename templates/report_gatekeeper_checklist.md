# Report Gatekeeper Checklist

## Required Sections

- [ ] 一句话结论
- [ ] 报告准出状态
- [ ] 决策权重表
- [ ] 数据可信度总评
- [ ] 关键数据卡片
- [ ] 核心变量排序
- [ ] 宏观周期与资产轮动
- [ ] 行业位置
- [ ] 公司基本盘
- [ ] 估值与安全边际
- [ ] Price Level Engine 加减仓点位精算
- [ ] 真实持仓执行
- [ ] Bear Case 反向验证
- [ ] 四类判断
- [ ] 最终操作方案
- [ ] 最大风险
- [ ] 下次复盘条件

## Blocking Issues

- [ ] No data reliability review.
- [ ] No Bear Case.
- [ ] No account execution review.
- [ ] Strong action based on low-confidence data.
- [ ] Price level based only on a round number.
- [ ] Missing invalidation condition.

# Standard Research Report Template Selector

This file is a selector only. Do not fill this bilingual file as a final report.

For Chinese reports, use:

```text
templates/standard_research_report_zh.md
```

For English reports, use:

```text
templates/standard_research_report_en.md
```

Language rule:

```text
User asks in Chinese -> generate the Chinese report template.
User asks in English -> generate the English report template.
Explicit language request overrides auto-detection.
Final report headings must not be bilingual.
```

See also:

```text
docs/language_policy.md
docs/report_export.md
```
